# 🚀 Pipeline de Engenharia de Dados - Databricks

## 📊 Sobre o Projeto

Este projeto demonstra a construção de um pipeline de dados utilizando a arquitetura Medallion (Bronze, Silver e Gold).

## 🧱 Arquitetura

* Bronze: dados brutos
* Silver: dados tratados
* Gold: dados agregados para análise

## 🛠️ Tecnologias

* Databricks
* PySpark
* Delta Lake
* SQL

## 📈 Análises realizadas

* Faturamento por produto
* Faturamento por categoria
* Evolução temporal das vendas

Demonstrar habilidades práticas em engenharia de dados, construção de pipelines e análise de dados.
